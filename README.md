## ğŸ“ Contributing

Have a recipe suggestion or improvement? Here's how to contribute:

### Using GitHub Desktop

1. **Fork the repository** on GitHub (click the Fork button)
2. **Clone your fork** using GitHub Desktop (see steps above)
3. **Create a new branch**:
   - In GitHub Desktop, click **Current Branch** â†’ **New Branch**
   - Name it something descriptive like `add-chocolate-chip-cookies`
4. **Add your recipe**:
   - Create a new `.txt` file with your recipe
   - Save it in the repository folder
5. **Commit your changes**:
   - GitHub Desktop will show your new file
   - Write a commit message like "Add chocolate chip cookie recipe"
   - Click **Commit to [your-branch-name]**
6. **Push to GitHub**:
   - Click **Push origin** in GitHub Desktop
7. **Create a Pull Request**:
   - Click **Create Pull Request** in GitHub Desktop
   - Fill out the PR description on GitHub
   - Submit for review!

### Using Command Line

1. Fork this repository
2. Create a new branch: `git checkout -b add-your-recipe`
3. Add your recipe as a new `.txt` file
4. Commit your changes: `git commit -am "Add new recipe"`
5. Push to your fork: `git push origin add-your-recipe`
6. Submit a pull request
